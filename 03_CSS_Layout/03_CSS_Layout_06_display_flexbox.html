CSS Layout - display: flex

flexbox

flexbox는 CSS 가변 상자 레이아웃(CSS Flexible Box Laybout) 모듈을 뜻한다.

사용자 인터페이스 디자인과 단방향 레이아웃에 최적화된 CSS 모듈이다.


부모 요소에 display: flex;속성을 적용하면 모든 자식 요소들은 flex 속성을 지닌다.
