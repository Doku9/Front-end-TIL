CSS Layout - display: inline

display: inline

inline 속성값은 새 줄에서 시작되지 않으며, 사용 가능한 화면 너비를 차지한다.

기본값으로 display: block;이 지정되어 있는 요소(<div>, <section>, <p> 등)에 사용할 경우 이어 붙일 수 있다.
