CSS Layout - float

float

grid에서는 열, 행 및 간격으로 구성되며, 간격은 거터(gutter)라고 한다.

flexbox는 일차원 레이아웃을 지원하지만, grid는 이차원 레이아웃을 지원한다.

즉 flexbox는 행을 변경 및 지정하며, grid에서는 행과 열을 변경 및 지정할 수 있다.
